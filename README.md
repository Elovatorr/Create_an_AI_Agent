# AI Coding Agent

This project features an AI coding agent designed to interact with a file system and execute Python code. The agent can list directory contents, read file content, write to files, and run Python scripts with specified arguments. This enables it to perform a variety of coding tasks, from simple file management to more complex code execution and debugging.

The AI agent operates by interpreting user requests and translating them into a series of file system operations or Python script executions. It provides a robust interface for automating development workflows, assisting with code generation, and managing project files efficiently. Its capabilities are confined to the provided toolset, ensuring secure and predictable interactions within the designated working directory.
